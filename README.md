# SentinelField Security Auditor

**SentinelField** is a professional-grade, native Windows security auditing utility designed for field operations and air-gapped environments. It is a single-file executable that requires no installation and zero cloud connectivity.

### Security Hardening Shield
Automated detection and remediation of critical vulnerabilities:
-   **RDP & SMBv1 Prevention**: One-click disabling of legacy protocols.
-   **Account Security**: Verification of Guest account status and LSA Protection.
-   **Credential Guard**: Audits system readiness for advanced credential protection.

### Network Sentry
Low-level visibility into your system's network activity:
-   **Process Mapping**: Maps every listening port directly to its Process ID (PID).
-   **Active Connections**: Real-time filterable grid of external connections.

### Field Compliance Auditor
Ensures workstation security for unattended environments:
-   **Screen Lock Verification**: Audits timeout policies and password protection.
-   **USB Data Control**: Checks Write Protection status on removable storage.

### System Diagnostics & Recovery
-   **Resolution Emergency**: Clears corrupted Windows Graphics Driver configuration caches to fix multi-monitor display issues on the fly.
